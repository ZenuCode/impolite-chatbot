# 불친절한 챗봇

물어보거나 대화하면 답변을 불친절하게 하는 Chatbot

사용한 기술
- ChatGPT API
- @chatscope/chat-ui-kit-react


개인 Open AI API Key를 .env에 VITE_API_KEY로 입력하시면 됩니다
